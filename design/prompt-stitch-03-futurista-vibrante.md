# Arvixy — Prompt Stitch IA · Futurista vibrante

Diseñar una landing page corporativa para “Arvixy”, empresa de desarrollo de software, automatización de procesos y consultoría tecnológica. La interfaz debe tener un estilo futurista, vibrante, creativo y visualmente impactante, con identidad claramente tecnológica. Usar una base oscura o degradada con colores intensos como azul eléctrico, cyan, violeta y magenta, combinados con tipografía moderna de estilo tech. Incorporar glow effects sutiles, glassmorphism, retículas digitales, nodos conectados, líneas abstractas, orbes de luz, dashboards flotantes y elementos visuales inspirados en inteligencia digital, automatización y sistemas avanzados. La página debe sentirse premium, innovadora, dinámica y memorable, manteniendo excelente usabilidad y una experiencia 100% responsive mobile-first.

## Secciones requeridas

- Header con navegación y CTA
- Hero principal de alto impacto
- Qué hacemos
- Servicios
- Beneficios
- Proceso de trabajo
- Casos de uso
- Tecnologías
- Sobre Arvixy
- Diferenciales
- CTA final
- Contacto con formulario
- Footer

## Indicación adicional

La interfaz debe transmitir visión de futuro, velocidad, inteligencia, creatividad y alto nivel técnico.
